## 一、本章诉求

修改智能体对话（web控制手机）请求为异步响应式接口，实时反馈到用户端（web）当前的执行动作。那么，这里要把 `Response<ChatResponseDTO>` 调整为 `ResponseBodyEmitter`

>**`ResponseBodyEmitter`** 是 Spring Framework（具体来说是 Spring Web MVC）中用于处理异步 HTTP 请求的一个核心类。它的主要作用是**允许服务端在一个 HTTP 请求的生命周期内，异步地、多次向客户端发送数据**。
